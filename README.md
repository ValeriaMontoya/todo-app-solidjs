# Solid Todo App

A modern todo application built with SolidJS and Tailwind CSS as part of my learning journey in web development.

## Features

- Todo management with add/remove/toggle functionality
- Editable todo items with double-click
- Todo persistence using localStorage
- Separated todo list into "To do" and "Completed" sections
- Completed task counter
- Dark/light theme toggle with localStorage persistence
- Responsive UI

## Tech Stack

| Layer           | Tools                                    |
| --------------- | ---------------------------------------- |
| Package manager | [pnpm](https://pnpm.io/)                 |
| Bundler         | [Vite](https://vite.dev/)                |
| Library         | [SolidJS](https://www.solidjs.com/)      |
| Styling         | [Tailwind CSS](https://tailwindcss.com/) |

## Getting Started

To run this project locally:

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Start the development server: `pnpm run dev`
